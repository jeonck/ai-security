이 섹션은 [학습 로드맵](/docs)에서 다루는 이론과 실습을 직접 손으로 확인할 때 사용할 수 있는 **오픈소스 도구**와 **공식 표준·참고 문서**를 한 곳에 모아둔 참고 자료실입니다.

`/docs`의 각 페이지는 "왜, 무엇을" 설명하는 데 집중하고, 이 섹션은 "어떤 도구로, 어디서 더 읽을 수 있는가"에 집중합니다. 즉 `/docs`를 읽다가 "이걸 직접 실행해보려면 뭘 설치해야 하지?" 또는 "이 표준의 원문은 어디서 볼 수 있지?"라는 질문이 떠오를 때 이 섹션으로 돌아오면 됩니다.

{{< callout type="info" >}}
직접 손으로 실습해보고 싶다면 [실습 (Labs)](/labs) 섹션도 함께 참고하세요. 이 섹션은 "도구 자체에 대한 설명"에 집중하고, `/labs`는 "그 도구를 사용한 단계별 실습"을 제공합니다.
{{< /callout >}}

## 사용 방법

1. [학습 로드맵](/docs)을 따라가며 개념을 먼저 익힙니다.
2. 해당 주제와 관련된 도구가 등장하면 이 섹션에서 설치 방법과 기본 사용법을 확인합니다.
3. [실습 (Labs)](/labs) 섹션에서 실제 코드를 따라 실행해봅니다.
4. 결과를 정리할 때 [표준 및 참고 문서](standards-references)의 공식 자료를 인용하여 보고서의 신뢰도를 높입니다.

{{< callout type="warning" >}}
이 섹션에 소개된 도구와 표준은 모두 외부 오픈소스 프로젝트 또는 공식 기관의 자료입니다. 버전, URL, 라이선스는 시간이 지나면 바뀔 수 있으니 링크를 통해 항상 최신 정보를 직접 확인하세요.
{{< /callout >}}
